# C# Programming – Warsaw University of Technology (Programowanie 3)

This repository contains a collection of C# code developed during the **"Programowanie 3"** course at the **Warsaw University of Technology (WUT)**.

## 📚 About the Course

The course focuses on advanced aspects of **C# programming** and **.NET platform**, with emphasis on:

- Object-Oriented Programming (OOP)
- Code organization and architecture
- Reflection and assemblies
- Advanced language features
- Writing clean and maintainable code

The exercises were implemented as part of laboratory assignments.

## 🧠 Topics Covered

The repository includes implementations and experiments related to:

- Classes, interfaces, and inheritance
- Encapsulation and abstraction
- Polymorphism
- Exception handling
- Delegates and events
- LINQ (Language Integrated Query)
- Reflection and metadata inspection
- Working with assemblies
- Basic design patterns (depending on lab)

These concepts are fundamental in C# and modern software development  [oai_citation:0‡Microsoft Learn](https://learn.microsoft.com/pl-pl/dotnet/csharp/fundamentals/tutorials/oop?utm_source=chatgpt.com).
